
# Article Platform

A full-stack web application where users can write, publish, read, like, and comment on articles. The platform works like a simplified social media system where content is shared and interacted with in real time.

---

##  Features

- Users can create and publish articles  
- Articles can be read by all users  
- Like system for engagement  
- Comment system for interaction  
- Dynamic and responsive UI  
- Real-time data handling with Firebase  
- Backend support using Flask  

---

##  Technologies Used

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Python  
- Flask  

### Database / Backend Services
- Firebase (Authentication / Database / Real-time data handling)



## Project Purpose

This project was developed to understand full-stack web development, including frontend design, backend logic, and database integration. It simulates a basic content-sharing platform where users can interact with posts in real time.



##  What I Learned

- Building dynamic web applications using Flask  
- Connecting frontend with backend APIs  
- Working with Firebase for real-time database operations  
- DOM manipulation and event handling in JavaScript  
- Structuring a full-stack project properly  
- Handling user interactions like likes and comments
